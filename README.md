# Python豆瓣图书数据可视化分析系统 python144

本项目是一个基于Python的豆瓣图书数据可视化分析系统，旨在通过对豆瓣图书数据的抓取和分析，为用户提供一个全面了解图书市场趋势和用户评价的平台。

## 技术栈

- **后端框架**: Django
- **数据库**: MySQL
- **前端技术**: HTML, CSS, JavaScript

## 功能模块

- **图书数据爬取**: 自动从豆瓣获取图书数据。
- **书籍分析**: 对图书的各类信息进行统计分析。
  - **种类分析**: 分析图书的分类分布。
  - **信息分析**: 对图书的详细信息进行挖掘。
  - **评论分析**: 对用户的评论数据进行情感分析等。
  - **年份分析**: 分析图书出版年份的分布情况。
  - **标题词云**: 显示图书标题的关键词分布。
  - **简介词云**: 显示图书简介中的关键词分布。
  - **图书推荐**: 根据用户行为和偏好推荐图书。

## 系统角色

- **用户**: 使用系统进行图书数据查询和分析。
- **管理员**: 负责系统维护和数据更新。
